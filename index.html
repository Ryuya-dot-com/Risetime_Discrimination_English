Listening Task 4 | Instructions

How this test works

Each trial plays three sounds; one of them differs from the other two.

The odd sound is always in position 1 or 3. Choose which one is different.

Start with practice using widely spaced stimuli to get used to the procedure.

Practice is 5 trials with feedback. When finished, press Space or use the button to begin the main run.

After 5 practice trials, start the main run with Space or the button.
